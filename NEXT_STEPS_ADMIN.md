# Next Steps for Admin - Product Updates

**Status:** Code changes complete ✅ | Ready for admin to update products

## 📋 What You Need to Do

### 1. Update Product Names to Standardized Format

**Current Format (Old):**
```
"Dubai Borka with Stone Work"
"Jumeirah Borka - Viral Collection"
"Nida Fabric Borka - Lightweight"
```

**New Format (Standardized):**
```
"Stone Embellished Classic Borka"
"Trending Collection Premium Borka"
"Feather Light Summer Borka"
```

**How to Update:**
1. Go to Admin → Products
2. Click Edit on each product
3. Update the title to standardized format
4. Save changes

---

### 2. Assign Correct Fabric to Each Product

**Available Fabrics (8 Total):**
1. Aroya Premium
2. Zoom Embroidery
3. Dubai Cherry
4. Bhabla Blend
5. Jacket Weave
6. Nida Premium
7. Japran Silk
8. Elix Georgette

**How to Assign:**
1. Go to Admin → Products → Edit
2. Find "Fabric" dropdown (now standardized)
3. Select the correct fabric from the list
4. Save changes

**Matching Guide:**
- Stone work products → Aroya Premium
- Embroidered products → Zoom Embroidery
- Printed products → Dubai Cherry or Jacket Weave
- Lightweight products → Nida Premium
- Elegant/formal → Elix Georgette
- Silk-like → Japran Silk
- Traditional → Bhabla Blend

---

### 3. Ensure All Products Have Sizes

**Required Sizes:**
- 38, 40, 42, 44, 46, 48, 50, 52, 54, 56, 58, 60

**How to Add Sizes:**
1. Go to Admin → Products → Edit
2. Find "Sizes" section
3. Select available sizes for the product
4. Save changes

**Important:** 
- Customers MUST select a size before adding to cart
- If product has no sizes, it can be added directly
- Most Borka products should have multiple sizes

---

### 4. Verify Product-Fabric Matching

**Check Each Product:**
- [ ] Product name is standardized
- [ ] Fabric is assigned correctly
- [ ] Fabric matches product type
- [ ] Sizes are selected
- [ ] Product displays correctly on shop

**Example Verification:**
```
Product: "Stone Embellished Classic Borka"
Fabric: Aroya Premium ✅ (matches stone work)
Sizes: 46, 48, 50, 52 ✅ (has sizes)
Status: Ready ✅
```

## 🎯 Step-by-Step Process

### For Each Product:

**Step 1: Open Product**
1. Go to Admin Dashboard
2. Click "Products"
3. Click "Edit" on product

**Step 2: Update Name**
1. Find "Product Title" field
2. Change to standardized format
3. Example: "Stone Embellished Classic Borka"

**Step 3: Select Fabric**
1. Find "Fabric" dropdown
2. Select from 8 options
3. Match fabric to product type

**Step 4: Add Sizes**
1. Find "Sizes" section
2. Select available sizes (38-60)
3. Most products should have 4-6 sizes

**Step 5: Save**
1. Click "Save" or "Update Product"
2. Verify changes on shop

## 🔍 Common Issues & Fixes

### Issue: Fabric dropdown not showing
**Fix:** Refresh page or clear browser cache

### Issue: Product not appearing on shop
**Fix:** Check if category is active and product is published

### Issue: Size not showing in filter
**Fix:** Ensure product has sizes selected

### Issue: Wrong fabric assigned
**Fix:** Edit product and select correct fabric from dropdown
